# README #

### Dependencies ###

* Node 7.2.0
* Npm 4.0.5
* Ionic 2.1.14

After install run:
$ ionic serve

Or to create a starter app add:
$ ionic start MyIonic2Project tabs --v2

Their docs are way better than this!
* https://ionicframework.com/docs/v2/setup/tutorial/
